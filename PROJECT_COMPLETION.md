## 🚀 How to Use

### Access the Website
```
URL: http://localhost/BE/Bose Enterprises/
```

### Navigate Pages
- Use the navigation menu
- Or use query parameters: `?page=home`, `?page=about`, etc.

### Customize Content
1. Edit page files in `app/views/pages/`
2. Update styling in `public/assets/css/style.css`
3. Modify configuration in `config/config.php`

### Add New Pages
1. Create new PHP file in `app/views/pages/`
2. Include components (head-meta, navbar, footer, footer-scripts)
3. Access via `?page=filename`
