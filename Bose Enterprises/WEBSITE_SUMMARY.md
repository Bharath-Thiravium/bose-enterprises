## 🔧 Customization Guide

### Update Company Information
Edit `config/config.php`:
```php
define('APP_NAME', 'BOSE ENTERPRISES');
define('APP_URL', 'http://localhost/BE/Bose Enterprises');
define('MAIL_FROM', 'info@bose-enterprises.in');
```

### Update Page Content
Edit individual page files in `app/views/pages/`:
- Modify text content
- Update service descriptions
- Change team member information
- Update contact details

### Update Styling
Edit `public/assets/css/style.css`:
- Modify CSS variables for colors
- Adjust animations and transitions
- Update responsive breakpoints

### Add New Pages
1. Create new PHP file in `app/views/pages/`
2. Include components (head-meta, navbar, footer, footer-scripts)
3. Access via `?page=filename`
