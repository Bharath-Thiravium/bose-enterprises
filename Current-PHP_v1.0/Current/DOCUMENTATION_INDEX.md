## Important Notes

1. **Do NOT change core template structure**
   - Only replace content
   - Keep component architecture

2. **Update all include paths**
   - From: `<?php include('partials/...'); ?>`
   - To: `<?php include('../components/...'); ?>`

3. **Maintain responsive design**
   - Bootstrap grid must work
   - Test on mobile devices

4. **Keep animations active**
   - GSAP and AOS libraries
   - Smooth transitions

5. **Test thoroughly**
   - All pages
   - All links
   - All forms
   - Responsive design
