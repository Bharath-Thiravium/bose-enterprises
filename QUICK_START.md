## 🔧 Customization

### Update Company Information
Edit `config/config.php` to change:
- Company name
- Website URL
- Email address
- Database settings

### Update Page Content
Edit files in `app/views/pages/`:
- `home.php` - Homepage content
- `about.php` - About page content
- `services.php` - Services page content
- `team.php` - Team page content
- `contact.php` - Contact page content

### Update Styling
Edit `public/assets/css/style.css` to change:
- Colors (CSS variables at top)
- Fonts and typography
- Animations and transitions
- Responsive breakpoints
